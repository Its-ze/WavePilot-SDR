Native RTL-SDR scanner for Windows and Linux

WavePilot SDR

Install the desktop receiver app, bind your RTL-SDR to the right driver, then use grouped receiver controls to scan preset channels, listen live, transcribe clear analog speech, and update from inside the app.

GitHub Repo Driver Setup

Install

Windows PowerShell

Set-ExecutionPolicy -Scope Process Bypass -Force
git clone https://github.com/Its-ze/WavePilot-SDR.git
cd WavePilot-SDR
.\scripts\install-windows.ps1
.\scripts\start-windows.ps1

The installer can download public RTL-SDR runtime DLLs, launch Zadig for the WinUSB driver step after you approve it, and add a desktop shortcut.

Linux

git clone https://github.com/Its-ze/WavePilot-SDR.git
cd WavePilot-SDR
chmod +x scripts/*.sh
./scripts/install-linux.sh --udev
./scripts/start-linux.sh

The installer can use your package manager for RTL-SDR libraries, including Noble/24.04 systems that ship librtlsdr2, and add udev rules for user-level USB access.

What You Get

Presets

Weather, airband, FM, marine, ham, FRS/GMRS, MURS, business, data, and oddities.

Scan Mode

Checks preset groups and ranks active signals by measured RF strength.

Listen Live

Select a preset or scan result to tune and start continuous WFM, NFM, or AM audio with optional mute and squelch.

Live Transcript

Uses the local Vosk model to transcribe clear analog speech from the active receiver stream.
