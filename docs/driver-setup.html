Driver Setup

WavePilot SDR can install the desktop app without administrator rights. USB driver binding still needs your approval because it changes how Windows or Linux exposes the SDR dongle.

Windows

.\scripts\install-windows.ps1 -InstallDriver
  1. Approve the Zadig administrator prompt.
  2. Open Options and enable List All Devices.
  3. Select RTL2838UHIDIR, Bulk-In Interface 0, or the matching Realtek RTL-SDR entry.
  4. Choose WinUSB.
  5. Install or replace the driver, then unplug and replug the dongle.

The runtime DLLs come from public RTL-SDR Blog Windows releases. Zadig comes from public libwdi/Zadig releases.

Linux

./scripts/install-linux.sh --udev

The installer can add udev rules for common Realtek RTL-SDR devices and install distro packages such as rtl-sdr plus the available librtlsdr package. On Pop!_OS / Ubuntu Noble that package is usually librtlsdr2; older Ubuntu/Debian releases may still use librtlsdr0. Unplug and replug the dongle after the udev step.
